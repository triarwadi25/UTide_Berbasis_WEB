# UTide_Berbasis_WEB
Source code aplikasi analisis pasang surut berbasis web yang dikembangkan menggunakan Python, Streamlit, dan pustaka UTIDE (Unified Tidal Analysis and Prediction). Aplikasi ini digunakan untuk analisis harmonik, prediksi pasang surut, penentuan tipe pasang surut, serta visualisasi hasil analisis.

Cara menggunakannya :

1. Buka Software Python
2. Pilih Directory yang digunakan dilaptop anda
3. Install Library yang dibutuhkan : pip install streamlit pandas numpy matplotlib utide scikit-learn openpyxl
4. Masukkan script berikut untuk menjalankanya : python -m streamlit run Pasang_Surut_UTide.py
